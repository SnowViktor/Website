My personal digital garden and professional portfolio, built with **Astro 6**, powered by **MDX**, and featuring mathematical typesetting.

## ✨ Features & Tech Stack

- **Core:** Astro v6 + MDX for fully interactive, content-driven pages.
- **Math:** Native LaTeX formulas rendering (`remark-math`, `rehype-katex`).
- **SEO & Perf:** Automated sitemap, RSS, viewport prefetching, and asset optimization.
- **Styling:** Vanilla CSS design system with responsive, premium layouts.
- **Deployment:** Vercel

## 🚀 Getting Started

**Prerequisites:** Node.js `v24.x+`, pnpm `v10.x+`.

```bash
git clone https://github.com/SnowViktor/Website.git && cd Website
pnpm install
pnpm run dev
```

### 📖 Key Commands

| Command | Description |
| :--- | :--- |
| `pnpm run dev` | Local dev server at `localhost:4321`. |
| `pnpm run build` | Full type-check and production build. |
| `pnpm run project:new` | Interactive CLI to generate a new article. |
